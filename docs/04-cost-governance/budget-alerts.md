# Budget + alerts setup

## Budget configuration (baseline)
1) Open Azure Portal -> Cost Management + Billing -> Budgets.
2) Create a monthly budget scoped to the subscription or rg-clopr2-katar711-gwc.
3) Set thresholds (example: 50%, 80%, 100%) and add notification emails.
4) Save and verify the budget appears in the list.
